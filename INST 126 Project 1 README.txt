This is a project that performs the four basic operations. These are addition, multiplication, subtraction, and division. These operations are represented as 1(addition), 2(subtraction), 3(multiplication), and 4(division). The user is first asked to choose which operation they would like to use by inputting the corresponding number 1-4. The user is then asked to input the first number and the second number to be used in the operation. Lastly, the answer to the equation is given as a number.
